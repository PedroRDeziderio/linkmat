1

Dada a Função F(x) = 3x - 7 determine:


A) F(0) = -7
F(0) = 3·0 - 7
3·0 = 0
F(0) = 0 - 7 = -7
B) F(1) = -4
F(1) = 3·1 - 7
3·1 = 3
F(1) = 3 - 7 = -4
C) F(-2) = -13
F(-2) = 3·(-2) - 7
3·(-2) = -6
F(-2) = -6 - 7 = -13

Determine os zeros ou raízes das funções:


A) f(x) = 4x - 8
4x - 8 = 0
4x = 8
x = 8 ÷ 4
x = 2
Raiz = 2
B) g(x) = -2x - 20
-2x - 20 = 0
-2x = 20
x = 20 ÷ (-2)
x = -10
Raiz = -10
C) H(x) = 8x + 10
8x + 10 = 0
8x = -10
x = -10 ÷ 8
x = -5/4 = -1,25
Raiz = -1,25

Faça o esboço dos gráficos das funções:


A) f(x) = 2x - 8
Ponto no eixo y: y = -8
Ponto no eixo x: 0 = 2x - 8 → 2x = 8 → x = 4
Função crescente
B) g(x) = 3x - 6
Ponto no eixo y: y = -6
Ponto no eixo x: 0 = 3x - 6 → 3x = 6 → x = 2
Função crescente
C) H(x) = -4x - 4
Ponto no eixo y: y = -4
Ponto no eixo x: 0 = -4x - 4 → -4x = 4 → x = -1
Função decrescente

Um plano telefônico custa R$ 50,00 ao mês, com franquia de 300 minutos, e cada minuto utilizado além da franquia custa R$ 0,60. Se a conta de um usuário, em determinado mês foi de R$ 125,00, quantos minutos foram utilizados neste mês?

Opções: a) 410 min   b) 415 min   c) 420 min   d) 425 min   e) 430 min


Fórmula: F(x) = 0,60x + b
x = 300 (franquia)
F(300) = 50 (valor base)
50 = 0,60·300 + b
50 = 180 + b
b = 50 - 180
b = -130
Agora, para F(x) = 125:
125 = 0,60x - 130
125 + 130 = 0,60x
255 = 0,60x
x = 255 ÷ 0,60
x = 425 minutos

Um taxista cobra R$ 5,00 de bandeirada (valor fixo) e R$ 2,50 por km rodado. A função que representa o preço para uma distância em km é:

P(d) = 2,5d + 5

a) Qual o preço para uma corrida de 12 km?
b) Qual a distância para pagar exatamente R$ 30?
c) Monte o gráfico.

Resolução:

a)
P(12) = 2,5(12) + 5 = 30 + 5 = 35

b)
30 = 2,5d + 5
2,5d = 25
d = 10

c)
Pontos para o gráfico:
Reta crescente que começa no ponto (0,5).

Uma fábrica tem custo fixo de R$ 500,00 e cada camiseta produzida custa mais R$ 8,00. A função do custo total para camisetas é:

C(q) = 8q + 500

a) Calcule o custo para 100 camisetas.
b) Quantas camisetas podem ser feitas com R$ 1.500?
c) Monte o gráfico.

Resolução:

a)
C(100) = 8(100) + 500 = 800 + 500 = 1300

b)
1500 = 8q + 500
8q = 1000
q = 125

c)
Pontos para o gráfico:
Reta crescente que começa no ponto (0,500).
